# Radxa Compliance Archive

This repository hosts public compliance documents (CE, FCC, RED, etc.) for Radxa products.

## Directory Layout

```
rock5/
  rock5a/
    ROCK5A_CE-EMC_Report.pdf
    ROCK5A_CE-EMC_Declaration.pdf
    ROCK5A_CE-EMC_Certificate.pdf
    ROCK5A_FCC_sDoC_Report.pdf
    ROCK5A_FCC_sDoC_Declaration.pdf
    ROCK5A_FCC_sDoC_Certificate.pdf
```

Add new products under their respective series directory (e.g., `rock5/rock5b/`). File names should clearly describe the product and certificate type.
